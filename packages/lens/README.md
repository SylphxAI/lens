# @sylphx/lens

Umbrella package that re-exports all Lens core functionality.

## Installation

```bash
bun add @sylphx/lens
```

## Usage

```typescript
import { t, entity, query, mutation, router } from "@sylphx/lens";

// All core exports available from single package
```

## Packages

- `@sylphx/lens-core` - Core schema types and utilities
- `@sylphx/lens-client` - Type-safe client with real-time subscriptions
- `@sylphx/lens-server` - Server runtime with WebSocket support
