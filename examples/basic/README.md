# Lens Basic Example

A simple example demonstrating the Lens API framework.

## Running

```bash
bun run dev
```

## Testing

```bash
bun test
```

## Structure

- `src/server.ts` - Server with router definition
- `src/index.ts` - Interactive demo (runs queries/mutations)
- `src/index.test.ts` - Integration tests
